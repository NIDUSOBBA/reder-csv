# Суть задачи
    1.Читает список покупок из файла
    2.Считает итоговую сумму со скидками
    3.Записывает готовый счёт в другой файл
# Входные данные
Файл products.csv в формате разделить можеть быть ( ; , | \t )
text:

    Название;Количество;Цена
    Milk;2;79.90
    Bread;1;45

Обработка

Для каждой позиции:

    1.Промежуточная сумма = цена × количество
    2.Скидка 10% - если купили 3+ штуки
    3.Итог по позиции = промежуточная сумма - скидка
# Выходные данные
Файл bill.csv в формате:
text

    Название;Количество;Цена;БылаСкидка(0/1);ИтогоПоПозиции
    Milk;2;79.90;0;159.80
    ...
    TOTAL;;;;1234.56
    ERRORS;;;;2

Главные правила

    1.Только BigDecimal для денег (никаких double!)
    2.Округление до 2 знаков (копеек)
    3.Пропускать плохие строки но считать их количество
    4.Игнорировать комментарии (строки с #)

Пример как должно работать

Вход (products.csv):
text

    Milk;2;79.90
    Bread;1;45
    Eggs;12;6.50
    # comment
    Oil;3;110

Расчёт для Oil:

    Промежуточная сумма: 110 × 3 = 330
    Скидка (т.к. 3 штуки): 330 × 10% = 33
    Итог: 330 - 33 = 297

Выход (bill.csv):
text

    Milk;2;79.90;0;159.80
    Bread;1;45.00;0;45.00
    Eggs;12;6.50;1;70.20
    Oil;3;110.00;1;297.00
    TOTAL;;;;572.00
    ERRORS;;;;0
